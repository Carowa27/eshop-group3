## Assignment

The task is to create a static webshop using html and css.
This will be an intro to working with HTML, CSS, Scrum, Trello and Github.
The finished product will show what we have learned during the period.

### Task description

In the assignment, we will create:

- an business plan where we briefly describe the business. This should also include the target group.
- a graphic profile where we show what our goal is through color and shape from a UI/UX perspective.
- wireframes/mockups on each page
- plan the work using Trello and use Scrum
- each team member must create at least one html page
- all pages must be adapted for mobile, tablet and desktop

Other:

- may not use frameworks such as Bootstrap
- may use SASS/LESS if the entire group agrees to use these
- may use JavaScript
- must use Github PR
- must work according to Scrum
